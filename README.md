# Clarity

A premium, vision-first AI platform for legal and medical document analysis. Know exactly what you sign.

## Project Structure

- **frontend/**: The user interface built with HTML, Vanilla CSS, and modern JavaScript.
- **backend/**: Placeholder for the analysis engine and API integration.

## Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Design System

Inspired by high-end tech aesthetics (Cluely), utilizing the **Manrope** font and a vibrant sapphire-blue palette with glassmorphic accents and smooth scroll-triggered animations.
